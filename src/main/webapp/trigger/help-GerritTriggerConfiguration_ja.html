どのGerritプロジェクトでビルドをトリガするかを指定します。

少なくとも１つのGerritプロジェクトとブランチのパターンを指定する必要があります。 そして、好きなだけ指定することができます。

最初に左側のテキストフィールドにGerritプロジェクトの名前を入力します。 ドロップダウンメニューの"タイプ"にある３つの異なる方法で、名前のパターンを指定できます。

  • Plain: Gerritプロジェクトの正確な名前。大小文字を区別します
  • Path: ANTスタイルのパターン Ex: "**/base/*"
  • RegExp: 正規表現

次に、トリガするブランチの名前を指定してください。タイプとパターンは上記と同様です。 例えば、Gerritプロジェクト内の全てのブランチでトリガするには、以下のように指定できます。

  • タイプ: Path
  • パターン: **

"ブランチ追加"を押して、さらにブランチパターンを追加することができます。

また、オプションで特定のファイルパスまたはトピックを複数指定することができます。これにより、例えば パッチセットにある特定のフォルダ内のファイルが含まれている時だけトリガーする、チェンジが特定の トピックに属している時だけトリガーする、といったことが可能になります。指定には上記の「タイプ」 で指定したものと同じ構文が使用できます。

ファイルパスを指定しない場合は、すべてのファイルがトリガーの対象になります。（トピックも同様です）

例えば、任意のフォルダ階層にあるjavaフォルダより下の全てのファイルでトリガーするには、以下のように指定できます。

  • Type: Path
  • Pattern: **/java/**

「ファイルパス追加」もしくは「トピック追加」ボタンをクリックしてさらに指定を追加することができます。
