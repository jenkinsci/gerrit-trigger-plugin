Important note when using Replication:

Please note that you may require the View Drafts permission within Gerrit
in order to have this trigger work consistently.


This permission should be applied for the user that is connecting to Gerrit as
specified with the Gerrit Server configuration section.


If you do NOT have the permission defined, the Gerrit Trigger may not be able to
see certain events and therefore cause builds to timeout waiting for the replication
to complete.
