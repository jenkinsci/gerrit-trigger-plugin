Allows checking and running unreviewed patches when connection is established. If unreviewed patchsets have been created during Jenkins server down time, Jenkins will trigger "patchset created" events for those patches when the connection is re-established.

Caution!
  • Property supports only plain text formatted project patterns. Other type of project patterns will be ignored.
  • Property supports only "patchset created" event. Other types of event will be ignored.
