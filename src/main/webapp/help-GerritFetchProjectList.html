Defines when and how the project list is fetched from Gerrit.

This list is used for project auto completion in the project configuration.

Enable project auto completion

If checked, this Gerrit server will be used to update the project list for project name auto-completion.

If unchecked, disables fetching of project names, which also means no project name auto-completion will be available for users configuring Jenkins projects.

Refresh project list delay

Delay, in seconds, before the initial fetch of the project list from this Gerrit server.

Project list fetch interval

For Gerrit servers with version < v2.12. The interval between fetches of the project list.
